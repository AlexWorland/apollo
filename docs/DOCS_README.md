# Generated Documentation

This directory contains automatically generated Doxygen documentation for all three repositories in this project.

## Contents

- `apollo/` - Apollo (Sunshine) server documentation
- `moonlight-common-c/` - Moonlight Common C library documentation  
- `moonlight-qt/` - Moonlight Qt client documentation

## Generation

Documentation is automatically generated by GitHub Actions when:
- A pull request is opened
- A pull request is updated (synchronized)
- A pull request is reopened

The documentation is built using Doxygen and committed back to the PR branch automatically.

## Manual Generation

To generate documentation locally:

### Apollo
```bash
cd apollo
mkdir -p build-docs && cd build-docs
cmake .. -DBUILD_DOCS=ON
cmake --build . --target docs
# Documentation will be in build-docs/docs/html/
```

### Moonlight Common C
```bash
cd moonlight-common-c
mkdir -p build-docs && cd build-docs
cmake .. -DBUILD_DOCS=ON
cmake --build . --target docs
# Documentation will be in build-docs/docs/html/
```

### Moonlight Qt
```bash
cd moonlight-qt/docs
mkdir -p build-docs && cd build-docs
cmake .. -DBUILD_DOCS=ON
cmake --build . --target docs
# Documentation will be in build-docs/html/
```

## Viewing Documentation

Open `index.html` in any of the documentation directories in a web browser to view the generated documentation.
