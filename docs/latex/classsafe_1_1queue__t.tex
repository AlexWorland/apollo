class safe\+::queue\+\_\+t$<$ T $>$}
Thread-\/safe queue class. 

Provides a thread-\/safe FIFO queue with a maximum size limit. When the queue reaches maximum capacity, it automatically clears. Thread-\/safe\+: all operations are protected by internal mutex.
