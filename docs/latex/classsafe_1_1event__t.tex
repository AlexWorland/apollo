class safe\+::event\+\_\+t$<$ T $>$}
Thread-\/safe event/signal class. 

Provides a thread-\/safe mechanism for signaling events between threads. Supports waiting for events, timeout-\/based waiting, and can be stopped/reset. Thread-\/safe\+: all operations are protected by internal mutex.
