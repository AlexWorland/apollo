std\+::vector$<$std\+::string$>$ proc\+::ctx\+\_\+t\+::detached}



Detached commands that run independently of Sunshine. 

Some applications (e.\+g., Steam) either exit quickly or run indefinitely. Apps with normal child processes are handled by process grouping (wait\+\_\+all). Apps with indirect child processes (e.\+g., UWP) use auto-\/detach heuristic. For background processes that shouldn\textquotesingle{}t be managed, use detached commands.
