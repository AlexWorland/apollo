class util\+::uniq\+\_\+ptr$<$ T, D $>$}
Unique pointer with addressable pointer storage. 

Similar to std\+::unique\+\_\+ptr but allows getting the address of the pointer itself, which is useful for C APIs that require pointer-\/to-\/pointer arguments.
