class safe\+::shared\+\_\+t$<$ T $>$}
Thread-\/safe shared object manager. 

Manages a single instance of an object with reference counting. The object is constructed on first access and destructed when the last reference is released. Thread-\/safe\+: all operations are protected by internal mutex.
